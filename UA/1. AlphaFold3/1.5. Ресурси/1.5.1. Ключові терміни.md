## Метрики якості

| Термін | Розшифровка | Суть |
|---|---|---|
| **LDDT** | Local Distance Difference Test | Локальна якість структури (0–100) |
| **DockQ** | Docking Quality | Якість докінгу (0–1); >0.23 = правильно, >0.8 = дуже висока |
| **RMSD** | Root Mean Square Deviation | Середнє відхилення атомних позицій (Å) |
| **GDT** | Global Distance Test | Глобальна подібність структур |
| **iLDDT** | Interface LDDT | LDDT для міжланцюгових інтерфейсів |
| **ipTM** | Interface predicted TM-score | Впевненість у якості інтерфейсу |
| **pLDDT** | predicted LDDT | Передбачена локальна якість |
| **PAE** | Predicted Aligned Error | Матриця попарних помилок між сегментами |
| **TM-score** | Template Modeling score | Подібність топологій структур |

## Архітектурні терміни

| Термін | Суть |
|---|---|
| **Pairformer** | Новий основний блок AF3; замінює Evoformer |
| **Evoformer** | Основний блок AF2; обробляв MSA + пари |
| **MSA** | Multiple Sequence Alignment — вирівнювання споріднених послідовностей |
| **Diffusion Module** | Генерує координати атомів через денойзінг |
| **Crop size** | Максимальна кількість токенів у мінібатчі навчання |
| **pTM** | Передбачений TM-score для ланцюга |

## Хімічні терміни

| Термін | Суть |
|---|---|
| **SMILES** | Текстове представлення хімічної структури ліганду |
| **Ligand** | Мала молекула, що зв'язується з білком |
| **Glycosylation** | Приєднання вуглеводів до білка |
| **Holo structure** | Структура білка зі зв'язаним лігандом |
| **Apo structure** | Структура білка без ліганду |
| **Covalent modification** | Ковалентна хімічна модифікація залишку |
| **CCD** | Chemical Components Dictionary (PDB) |

## Бенчмарки

| Назва | Опис |
|---|---|
| **PoseBusters** | Набір з 428 (V1) або 308 (V2) білок-лігандних структур |
| **CASP15** | Critical Assessment of Structure Prediction, 15-й раунд |
| **CAID 2** | Оцінка передбачення невпорядкованих регіонів |
| **PDB** | Protein Data Bank — база 3D структур біомолекул |
