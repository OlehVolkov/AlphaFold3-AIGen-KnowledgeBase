## Що таке FASTA?

**FASTA** — текстовий формат для зберігання біологічних послідовностей (білки, ДНК, РНК). Названий на честь програми FASTA для порівняння послідовностей (1988). Є де-факто стандартом для обміну послідовностями між інструментами біоінформатики.

### Повний приклад запису

```text
>sp|P68871|HBB_HUMAN Hemoglobin subunit beta OS=Homo sapiens OX=9606 GN=HBB PE=1 SV=2
MVHLTPEEKSAVTALWGKVNVDEVGGEALGRLLVVYPWTQRFFESFGDLSTPDAVMGNPK
VKAHGKKVLGAFSDGLAHLDNLKGTFATLSELHCDKLHVDPENFRLLGNVLVCVLAHHFG
KEFTPPVQAAYQKVVAGVANALAHKYH
```

### Анатомія рядка заголовку (UniProt формат)

```text
>sp|P68871|HBB_HUMAN Hemoglobin subunit beta OS=Homo sapiens OX=9606 GN=HBB PE=1 SV=2
  ^^  ^^^^^^ ^^^^^^^^^^ ^^^^^^^^^^^^^^^^^^^^^^^^^ ^^^^^^^^^^ ^^^^^^^^ ^^^^ ^^^^
  |   |      |          |                         |          |        |    |
  db  AccID  EntryName  ProteinName               Organism   TaxonID  Gene PEvidence
```

| Поле | Значення | Опис |
|------|----------|------|
| `db` | `sp` / `tr` | База: SwissProt або TrEMBL |
| `AccID` | `P68871` | Унікальний акцесійний номер |
| `EntryName` | `HBB_HUMAN` | Назва\_Організм |
| `OS=` | `Homo sapiens` | Організм |
| `OX=` | `9606` | NCBI Taxonomy ID |
| `GN=` | `HBB` | Назва гену |
| `PE=` | `1` | Рівень доказу (1=підтверджено) |
| `SV=` | `2` | Версія послідовності |

### Однолітерні коди амінокислот

```text
A Ala  Аланін        C Cys  Цистеїн       D Asp  Аспартат
E Glu  Глутамат      F Phe  Фенілаланін   G Gly  Гліцин
H His  Гістидин      I Ile  Ізолейцин     K Lys  Лізин
L Leu  Лейцин        M Met  Метіонін      N Asn  Аспарагін
P Pro  Пролін        Q Gln  Глутамін      R Arg  Аргінін
S Ser  Серин         T Thr  Треонін       V Val  Валін
W Trp  Триптофан     Y Tyr  Тирозин

X  — невідома амінокислота
*  — стоп-кодон
-  — гап у вирівнюванні
```

### Коди нуклеотидів (ДНК/РНК)

```text
A  Аденін     C  Цитозин    G  Гуанін     T  Тимін (ДНК)
U  Урацил (РНК)
N  — будь-який нуклеотид
R  — A або G (пурини)
Y  — C або T (піримідини)
```

## Варіанти FASTA форматів

| Формат | Розширення | Опис |
|--------|-----------|------|
| FASTA | `.fasta`, `.fa` | Загальний формат |
| FASTQ | `.fastq`, `.fq` | FASTA + якість (NGS reads) |
| Multi-FASTA | `.fasta` | Кілька записів в одному файлі |
| Masked FASTA | `.fasta` | Малі літери = м'яке маскування (повтори) |
| Stockholm | `.sto` | Вирівнювання з анотацією |

## Читання FASTA — власноруч (без бібліотек)

```python
def parse_fasta(filepath: str) -> dict[str, dict]:
    """
    Читає FASTA файл, повертає {id: {seq, description}}.
    Обробляє багаторядкові послідовності та порожні рядки.
    """
    records = {}
    current_id = None
    current_desc = ""
    current_seq = []

    with open(filepath, "r") as f:
        for line in f:
            line = line.rstrip("\n")
            if not line.strip():
                continue                          # пропускаємо порожні рядки
            if line.startswith(">"):
                if current_id:
                    records[current_id] = {
                        "seq": "".join(current_seq),
                        "description": current_desc,
                    }
                header = line[1:]
                parts = header.split(None, 1)     # розбиваємо по першому пробілу
                current_id   = parts[0]
                current_desc = parts[1] if len(parts) > 1 else ""
                current_seq  = []
            else:
                current_seq.append(line.upper().strip())

    if current_id:                                # останній запис
        records[current_id] = {
            "seq": "".join(current_seq),
            "description": current_desc,
        }
    return records


records = parse_fasta("proteins.fasta")
for seq_id, data in records.items():
    print(f"{seq_id:20s}  {len(data['seq'])} aa  {data['description'][:40]}")
```

## Читання та запис через BioPython

```python
from Bio import SeqIO
from Bio.SeqRecord import SeqRecord
from Bio.Seq import Seq

# --- Читання ---
records = list(SeqIO.parse("proteins.fasta", "fasta"))

for rec in records:
    print(f"ID:          {rec.id}")
    print(f"Назва:       {rec.name}")
    print(f"Опис:        {rec.description}")
    print(f"Довжина:     {len(rec.seq)} aa")
    print(f"Послідовність: {rec.seq[:30]}...")
    print()

# --- Запис ---
new_records = [
    SeqRecord(Seq("MVHLTPEEKSAVTALWGKVNV"), id="HBB_HUMAN",
              description="Hemoglobin subunit beta OS=Homo sapiens"),
    SeqRecord(Seq("MGHFTEEDKATITSLWGKVN"),  id="HBA_HUMAN",
              description="Hemoglobin subunit alpha OS=Homo sapiens"),
]

with open("output.fasta", "w") as fh:
    SeqIO.write(new_records, fh, "fasta")

# --- Конвертація форматів ---
# FASTA → GenBank
SeqIO.convert("sequences.fasta", "fasta", "sequences.gb", "genbank")

# GenBank → FASTA (витягнути лише послідовності)
SeqIO.convert("sequences.gb", "genbank", "sequences.fasta", "fasta")
```

## Аналіз складу послідовності

```python
from Bio import SeqIO
from Bio.SeqUtils.ProtParam import ProteinAnalysis
from collections import Counter

rec = list(SeqIO.parse("proteins.fasta", "fasta"))[0]
seq_str = str(rec.seq)

# Амінокислотний склад
analysis = ProteinAnalysis(seq_str)
aa_count  = analysis.count_amino_acids()
aa_percent = analysis.get_amino_acids_percent()

print(f"Довжина: {len(seq_str)}")
print(f"Молекулярна маса: {analysis.molecular_weight():.1f} Da")
print(f"Ізоелектрична точка (pI): {analysis.isoelectric_point():.2f}")
print(f"GRAVY індекс (гідрофобність): {analysis.gravy():.3f}")
print(f"Нестабільний індекс: {analysis.instability_index():.1f}")
print()

# Топ-5 найчастіших амінокислот
top5 = sorted(aa_percent.items(), key=lambda x: -x[1])[:5]
for aa, pct in top5:
    print(f"  {aa}: {pct*100:.1f}%")
```

## Пошук мотивів у послідовності

```python
import re
from Bio import SeqIO

records = list(SeqIO.parse("proteins.fasta", "fasta"))

# Регулярні вирази для біологічних мотивів
MOTIFS = {
    "N-глікозилювання":   r"N[^P][ST][^P]",       # NxS/T (не перед Pro)
    "RGD (інтегрин)":     r"RGD",
    "KDEL (ретенція ER)": r"KDEL$",
    "Казеїнкіназа II":    r"[ST]..[DE]",
    "PKA сайт":           r"R[RK]..[ST]",
    "Ядерна локалізація": r"K[KR]{2,}",
}

for rec in records:
    seq = str(rec.seq)
    print(f"\n=== {rec.id} ({len(seq)} aa) ===")
    for name, pattern in MOTIFS.items():
        matches = [(m.start(), m.group()) for m in re.finditer(pattern, seq)]
        if matches:
            locs = ", ".join(f"{pos}:{m}" for pos, m in matches[:5])
            print(f"  {name:25s}: {locs}")
```

## Підготовка FASTA для AlphaFold 3

AF3 приймає мультиланцюговий FASTA, де кожен ланцюг — окремий запис.

```python
from Bio.SeqRecord import SeqRecord
from Bio.Seq import Seq
from Bio import SeqIO

def prepare_af3_fasta(
    chains: dict[str, str],
    output_path: str,
    copies: dict[str, int] | None = None,
):
    """
    Готує мультиланцюговий FASTA для AF3.

    chains  — {chain_id: sequence}
    copies  — {chain_id: кількість копій} для гомомерів
    """
    copies = copies or {}
    records = []

    for chain_id, seq in chains.items():
        n = copies.get(chain_id, 1)
        for i in range(n):
            label = f"{chain_id}_{i+1}" if n > 1 else chain_id
            records.append(SeqRecord(Seq(seq), id=label, description=""))

    with open(output_path, "w") as fh:
        SeqIO.write(records, fh, "fasta")
    print(f"Записано {len(records)} ланцюгів → {output_path}")


# Гетеродимер
prepare_af3_fasta({"A": "MVHLT...", "B": "MGHFT..."}, "dimer.fasta")

# Гомотетрамер (4 однакові ланцюги)
prepare_af3_fasta({"A": "MVHLT..."}, "tetramer.fasta", copies={"A": 4})
```

## Завантаження послідовності з UniProt / NCBI

```python
from Bio import Entrez, SeqIO
import urllib.request

# --- Варіант 1: UniProt REST API ---
def fetch_uniprot(accession: str) -> str:
    url = f"https://rest.uniprot.org/uniprotkb/{accession}.fasta"
    with urllib.request.urlopen(url) as resp:
        fasta = resp.read().decode()
    record = next(SeqIO.parse(fasta.splitlines(keepends=True), "fasta"))
    return str(record.seq)

seq = fetch_uniprot("P68871")   # Гемоглобін β
print(f"P68871: {len(seq)} aa")

# --- Варіант 2: NCBI Entrez ---
Entrez.email = "your@email.com"   # обов'язково!

def fetch_ncbi(accession: str) -> str:
    handle = Entrez.efetch(db="protein", id=accession,
                           rettype="fasta", retmode="text")
    record = SeqIO.read(handle, "fasta")
    handle.close()
    return str(record.seq)

seq_ncbi = fetch_ncbi("NP_000509.1")
print(f"NP_000509.1: {len(seq_ncbi)} aa")
```

## Фільтрація, дедуплікація, статистика

```python
from Bio import SeqIO
from collections import Counter

records = list(SeqIO.parse("large_db.fasta", "fasta"))

# Фільтрація
filtered = [r for r in records
            if 50 <= len(r.seq) <= 2000          # за довжиною
            and "X" not in str(r.seq)            # без невідомих aa
            and str(r.seq).startswith("M")]      # починається з Met

# Дедуплікація за точною послідовністю
seen, unique = set(), []
for r in filtered:
    key = str(r.seq)
    if key not in seen:
        seen.add(key)
        unique.append(r)

# Статистика
lengths = [len(r.seq) for r in unique]
print(f"Всього:     {len(records)}")
print(f"Після філ.: {len(filtered)}")
print(f"Унікальних: {len(unique)}")
print(f"Довжина:    min={min(lengths)}, max={max(lengths)}, "
      f"median={sorted(lengths)[len(lengths)//2]}")

# Запис результату
with open("filtered.fasta", "w") as fh:
    SeqIO.write(unique, fh, "fasta")
```
