# Точність по типах комплексів

[[🏠 Головна]] > Результати

## Білок–Ліганд (PoseBusters)

**Метрика:** % структур з pocket-aligned RMSD < 2 Å  
**Датасет:** 428 структур (PoseBusters V1, серпень 2023)

| Метод | Результат |
|---|---|
| **AF3 (2019 cutoff)** | **76.4%** |
| RoseTTAFold All-Atom | 42.0% |
| AutoDock Vina | ~52% |

- p = 2.27×10⁻¹³ (AF3 vs Vina)
- AF3 не використовує інформацію про кишеню (blind docking)
- Chirality violation rate: ~4.4% (відома проблема)

## Білок–Нуклеїнові кислоти

**Метрика:** Interface LDDT (iLDDT)

| Тип | AF3 | RoseTTAFold2NA |
|---|---|---|
| Білок–RNA | **39.4** | 19.0 |
| Білок–dsDNA | **64.8** | 28.3 |

Оцінка на структурах < 1000 залишків (обмеження RF2NA).

## RNA (CASP15)

AF3 перевищує RoseTTAFold2NA та AIchemy_RNA (кращий AI-метод на CASP15), але **не досягає** рівня AIchemy_RNA2 (з людським втручанням).

## Ковалентні модифікації

**Метрика:** % передбачень з pocket RMSD < 2 Å

| Тип | AF3 |
|---|---|
| Зв'язані ліганди | 78.5% |
| Глікозилювання (якісні) | 72.1% |
| Модифіковані залишки | 59.9% |
| Модифікована ДНК | 68.6% |

## Білок–Білок

**Метрика:** % DockQ > 0.23

| Тип | AF3 | AF-Multimer 2.3 |
|---|---|---|
| Усі білок-білок | **76.6%** | 67.5% |
| Антитіло–антиген | **62.9%** | 29.6% |
| Мономери (LDDT) | **86.9** | 85.5 |

### Антитіла — особливий випадок
Якість передбачення антитіл **продовжує зростати** зі збільшенням кількості seeds — навіть до 1000! (p = 2×10⁻⁵). Це не спостерігається для інших типів молекул.
