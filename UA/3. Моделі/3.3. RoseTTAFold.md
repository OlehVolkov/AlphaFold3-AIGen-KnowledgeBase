# 3.3. RoseTTAFold

[[UA/Головна]] > [[UA/3. Моделі/3.0. Огляд моделей|Моделі]] > RoseTTAFold
🇬🇧 [[EN/3. Models/3.3. RoseTTAFold|English]]

`RoseTTAFold` — нейромережева модель структури білків із триканальною інтеграцією 1D/2D/3D представлень.

## Коли корисна

- Практичний baseline для задач структурного передбачення.
- Альтернатива для незалежної перевірки результатів AF-пайплайнів.

## Обмеження

- У складних комплексах часто поступається новішим моделям.

> Baek et al. (2021). *Accurate prediction of protein structures and interactions using a three-track neural network*. Science.
> DOI: [10.1126/science.abj8754](https://doi.org/10.1126/science.abj8754)
