# 3.9. RoseTTAFoldNA

[[UA/Головна]] > [[UA/3. Моделі/3.0. Огляд моделей|Моделі]] > RoseTTAFoldNA
🇬🇧 [[EN/3. Models/3.9. RoseTTAFoldNA|English]]

`RoseTTAFoldNA` розширює сімейство `RoseTTAFold` від білків до нуклеїнових кислот і `protein–DNA` / `protein–RNA` комплексів. Це одна з найкорисніших focused-reference моделей, якщо базі знань потрібне сильніше покриття nucleic-acid-aware structure prediction.

## Чому RoseTTAFoldNA важливий

Передбачення protein-only structures покращилося значно раніше, ніж пряме моделювання nucleic-acid complexes.
Через це виник practical gap:

- передбачення protein monomers стало сильним;
- docking-style assembly для нуклеїнових комплексів лишався складним;
- `protein–DNA` і `protein–RNA` interfaces були слабким місцем.

RoseTTAFoldNA важливий саме тому, що націлений безпосередньо на цю прогалину.

## Головна ідея

Модель зберігає three-track логіку `RoseTTAFold`, але розширює всі основні представлення так, щоб вони включали нуклеїнові кислоти:

- додаткові 1D tokens для DNA і RNA residues;
- узагальнені 2D pair features для `protein–NA` і `NA–NA` interactions;
- узагальнений 3D coordinate reasoning для nucleotide geometry.

## Сильні сторони

| Сильна сторона | Чому це важливо |
| --- | --- |
| Прямий фокус на protein–NA | Концептуально краще підходить, ніж protein-only baselines |
| Confidence estimates | Корисно для відсіву правдоподібних interface predictions |
| Добре доповнює AF3 | Дозволяє порівнювати generalist і specialized reasoning |

## Коли RoseTTAFoldNA корисний

- коли задача головно про `protein–DNA` або `protein–RNA` комплекси;
- коли потрібен спеціалізований nucleic-acid-aware baseline;
- коли треба зрозуміти, чи failure пов'язаний із general complex modeling чи саме з NA-specific geometry.

## Обмеження

| Обмеження | Чому це важливо |
| --- | --- |
| Вужчий, ніж AF3 | Це спеціалізована, а не універсальна biomolecular model |
| Менш придатний для ligand-heavy задач | Його головна цінність це nucleic-acid interaction modeling |
| Складні кейси все ще складні | Великі гнучкі нуклеїнові кислоти та нестандартні interfaces і далі важкі |
