# 4.1. PDB

[[UA/Головна]] > [[UA/4. Датасети/4.0. Огляд датасетів|Датасети]] > PDB
🇬🇧 [[EN/4. Datasets/4.1. PDB|English]]

`PDB` (Protein Data Bank) — головний відкритий репозиторій експериментальних 3D-структур біомолекул.

## Ключові риси

- Експериментальні методи: X-ray, cryo-EM, NMR.
- Формати: `mmCIF`, legacy `PDB`.
- Широке покриття білків, комплексів, лігандів.

## Коли використовувати

- Ground truth для валідації моделей.
- Вибір референс-структур для порівняння.

> RCSB PDB (official portal): [https://www.rcsb.org](https://www.rcsb.org)
