# 4.2. UniProt

[[🏠 Головна]] > [[UA/4. Датасети/4.0. Огляд датасетів|Датасети]] > UniProt
🇬🇧 [[EN/4. Datasets/4.2. UniProt|English]]

`UniProt` — центральна база білкових послідовностей та функціональних анотацій.

## Ключові риси

- `UniProtKB/Swiss-Prot` (curated) та `UniProtKB/TrEMBL` (automated).
- ID-стандарти для білків та міжбазова інтеграція.
- API для програмного доступу.

## Коли використовувати

- Отримання FASTA для моделювання.
- Збагачення структурного аналізу функціональною анотацією.

> UniProt (official portal): [https://www.uniprot.org](https://www.uniprot.org)
