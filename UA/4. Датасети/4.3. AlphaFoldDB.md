# 4.3. AlphaFoldDB

[[UA/Головна]] > [[UA/4. Датасети/4.0. Огляд датасетів|Датасети]] > AlphaFoldDB
🇬🇧 [[EN/4. Datasets/4.3. AlphaFoldDB|English]]

> AlphaFold Protein Structure Database (2021, EMBL-EBI + DeepMind) — передбачені структури для практично всіх відомих білків основних модельних організмів і повного людського протеому.

---

## Масштаб і покриття

| Показник | Значення (2025) |
| --- | --- |
| Загальна кількість передбачених структур | ~200 000 000 |
| Покриття людського протеому | ~20 000 білків (майже повне) |
| Охоплені організми | >1 000 000 |
| Використана модель | AlphaFold2 (v4) |
| Метрика впевненості | pLDDT на залишок |
| Доступні формати | mmCIF, PDB, PAE JSON |

## Кольорова схема pLDDT

| Діапазон pLDDT | Колір | Інтерпретація |
| --- | --- | --- |
| > 90 | Темно-синій | Дуже висока впевненість |
| 70–90 | Блакитний | Впевнений |
| 50–70 | Жовтий | Низька впевненість, використовувати обережно |
| < 50 | Помаранчевий | Дуже низька впевненість, ймовірно невпорядкована ділянка |

## Версії

| Версія | Реліз | Ключові зміни |
| --- | --- | --- |
| v1 | Лип 2021 | 21 протеом, ~365K структур |
| v2 | Груд 2021 | 48 організмів, масштаб UniProt |
| v3 | Лип 2022 | ~200M структур, повний UniProt |
| v4 | Лист 2022 | Покращена модель + перезбирання |

## Роль у контексті AF3

AlphaFoldDB побудована на передбаченнях AF2. Це **не** навчальна вибірка для AF3, але слугує для:

| Використання | Деталі |
| --- | --- |
| Базове порівняння | AF3 vs AF2 на тих самих цілях |
| Джерело шаблонів | AF2 передбачення як структурні шаблони в деяких пайплайнах |
| Скринінг у масштабі протеому | Швидкий попередній фільтр перед коштовними запусками на сервері AF3 |
| Пошук ліків | Структурні гіпотези для цілей без записів у PDB |

## Переваги vs обмеження

| Переваги | Обмеження |
| --- | --- |
| Майже повне покриття протеому | Тільки AF2 — немає лігандів, ДНК/РНК, комплексів |
| Безкоштовний відкритий доступ через API | Впевненість ≠ біологічна значущість |
| pLDDT доступний на кожен залишок | Невпорядковані ділянки завжди мають низький pLDDT |
| Матриця PAE для аналізу меж доменів | Статичні передбачення, без конформаційних ансамблів |
| Пряма інтеграція з UniProt ID | Передбачення мультимерів не включені |

## AlphaFoldDB vs PDB

| Аспект | AlphaFoldDB | PDB |
| --- | --- | --- |
| Джерело | Обчислювальне (AF2) | Експериментальне |
| Покриття | ~200M послідовностей | ~220K структур |
| Точність | Висока для впорядкованих ділянок | Ground truth |
| Ліганди | ✗ | ✓ |
| Комплекси | ✗ | ✓ |
| Роздільна здатність | pLDDT як проксі | Å (атомна) |
| Відкритий доступ | ✓ | ✓ |
