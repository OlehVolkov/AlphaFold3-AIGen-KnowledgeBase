## Короткий огляд

**Дифузійні моделі** (DDPM) генерують дані через зворотній процес видалення шуму:

$$\boxed{q(x_t|x_0) = \mathcal{N}(\sqrt{\bar\alpha_t}\,x_0,\,(1-\bar\alpha_t)\mathbf{I})} \quad \longleftrightarrow \quad p_\theta(x_{t-1}|x_t) = \mathcal{N}(\mu_\theta,\Sigma_\theta)$$

## В контексті AlphaFold 3

AF3 генерує **атомні координати** $\mathbf{X}\in\mathbb{R}^{N_\text{atoms}\times3}$ через дифузію з conditioning від Pairformer (single $s$ + pair $z$). Це дозволяє моделювати весь молекулярний комплекс — білки, ДНК, РНК, ліганди — в одному процесі.

## U-Net для генерації зображень

`U-Net` (U-подібна мережа) — енкодер-декодер з `skip connections` між симетричними рівнями:

- **Encoder path**: зменшує просторову роздільність і збільшує число каналів.
- **Bottleneck**: містить глобальніші ознаки.
- **Decoder path**: відновлює роздільність.
- **Skip connections**: повертають деталі з ранніх шарів для точного відновлення структури.

У diffusion-моделях для зображень (DDPM/Latent Diffusion) U-Net параметризує $\varepsilon_\theta(x_t, t, c)$ або $v_\theta(\cdot)$.

## Порівняння: AF3 diffusion vs image diffusion (U-Net)

| Аспект | AF3 diffusion | Image diffusion (U-Net) |
|---|---|---|
| Простір стану | 3D координати атомів $\mathbb{R}^{N\times3}$ | Пікселі або latent-карти $\mathbb{R}^{H\times W\times C}$ |
| Що денойзимо | Геометрію молекулярного комплексу | Зображення/латент |
| Основна мережа | Геометрично-обумовлений модуль AF3 (з Pairformer-conditioning) | U-Net (encoder-decoder + skip connections) |
| Conditioning | Single/pair репрезентації, типи молекул, шаблони/контекст | Текст, клас, image-conditioning (через cross-attention) |
| Симетрії | Важлива $SE(3)$-узгодженість (оберт/зсув у 3D) | Трансляційна структура 2D-grid; повна обертальна еквіваріантність зазвичай не вбудована |
| Ціль навчання | Шум/score у координатному просторі + структурні обмеження | Шум $\varepsilon$, velocity $v$, або score у 2D/latent просторі |
| Вихід | 3D структура комплексу (білок/РНК/ДНК/ліганд) | Синтезоване зображення |
| Ключові метрики | RMSD, lDDT/pLDDT, DockQ, clash/contact quality | FID, CLIP score, людська оцінка якості |

Коротко: обидва підходи оптимізують зворотний процес дифузії, але **індуктивні упередження (inductive biases)** відрізняються: для AF3 критична 3D-геометрія та фізична узгодженість, для image diffusion — багатомасштабна 2D-структура і семантичний conditioning.
