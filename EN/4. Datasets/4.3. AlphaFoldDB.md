# 4.3. AlphaFoldDB

[[Home|Home]] > [[EN/4. Datasets/4.0. Datasets Overview|Datasets]] > AlphaFoldDB
🇺🇦 [[UA/4. Датасети/4.3. AlphaFoldDB|Українська]]

> AlphaFold Protein Structure Database (2021, EMBL-EBI + DeepMind) provides predicted structures for virtually every known protein across major model organisms and the full human proteome.

---

## Scale and coverage

| Metric | Value (2025) |
| --- | --- |
| Total predicted structures | ~200,000,000 |
| Human proteome coverage | ~20,000 proteins (near-complete) |
| Organisms covered | >1,000,000 |
| Model used | AlphaFold2 (v4) |
| Confidence metric | pLDDT per residue |
| Formats available | mmCIF, PDB, PAE JSON |

## pLDDT confidence color scheme

| pLDDT range | Color | Interpretation |
| --- | --- | --- |
| > 90 | Dark blue | Very high confidence |
| 70–90 | Light blue | Confident |
| 50–70 | Yellow | Low confidence, use with caution |
| < 50 | Orange | Very low confidence, likely disordered |

## Versioning

| Version | Release | Key changes |
| --- | --- | --- |
| v1 | Jul 2021 | 21 proteomes, ~365K structures |
| v2 | Dec 2021 | 48 organisms, UniProt-scale |
| v3 | Jul 2022 | ~200M structures, full UniProt |
| v4 | Nov 2022 | Improved model + reranking |

## Role in AF3 context

AlphaFoldDB is built on AF2 predictions. It is **not** a training set for AF3, but serves as:

| Use | Detail |
| --- | --- |
| Baseline comparison | AF3 vs AF2 accuracy on the same targets |
| Template source | AF2 predictions used as structural templates in some pipelines |
| Proteome-scale screening | Fast pre-filter before expensive AF3 server runs |
| Drug discovery | Structural hypotheses for targets without PDB entries |

## Strengths vs limitations

| Strengths | Limitations |
| --- | --- |
| Near-complete proteome coverage | AF2 only — no ligands, DNA/RNA, complexes |
| Free, open access via API | Confidence ≠ biological relevance |
| pLDDT available per residue | Disordered regions always low pLDDT |
| PAE matrix for domain boundary analysis | Static predictions, no conformational ensembles |
| Direct UniProt ID integration | Multimer predictions not included |

## AlphaFoldDB vs PDB

| Aspect | AlphaFoldDB | PDB |
| --- | --- | --- |
| Source | Computational (AF2) | Experimental |
| Coverage | ~200M sequences | ~220K structures |
| Accuracy | High for ordered regions | Ground truth |
| Ligands | ✗ | ✓ |
| Complexes | ✗ | ✓ |
| Resolution | pLDDT proxy | Å (atomic) |
| Free access | ✓ | ✓ |
